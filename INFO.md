--- Five Nights at Freddy's made using JavaScript, HTML5 and CSS by streetz ---
-------------------------------------------------------------------------------

FNaF JS is a small project I'm working on which recreates the original FNaF 1 with some additional gameplay changes made.

Trello where you can see the progress made on the project so far: https://trello.com/b/3maV07eF/fnaf-java-script
